Passion for Web Development

About Me


My name is Saulius Simkus. I am a Front-End Web Developer skilled in HTML, CSS, JavaScript, jQuery library, Bootstrap framework, WordPress CMS, version control system Git.

I can build websites from design making sure they are responsive and suitable for different screen sizes.

I very much enjoy learning and look forward to getting new skills every single day.

Although my previous positions involved sales, customer relationship management, client relations, I have always had interest in computers, smartphones, software, internet technologies.

Why Web Development?

In 2017, I wanted to create a website – my online photo gallery but did not want to use any preset templates. And I thought - “what if I created my web pages the way I wanted?“.

I started studying HTML, CSS, JavaScript, jQuery. The more I was getting into it, the more it got interesting.

Being a photography enthusiast I gained Adobe applications' skills which also helped me to develop an eye for design.

Learning web development became very natural to me and I love it!
